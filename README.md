# IPTV Justin

This repository is a simple IPTV bundle with a playlist, an EPG guide, and a basic landing page.

## Files
- Playlist: [channels.m3u](channels.m3u)
- EPG guide: [guide.xml](guide.xml)
- Web page: [index.html](index.html)
- Logos folder: [logos](logos)

## What this includes
- A combined M3U playlist for your channels
- An XMLTV/EPG file for compatible players
- A simple browser page that links to the playlist

## How to use it
1. Open [channels.m3u](channels.m3u) in an M3U-compatible player such as VLC, TiviMate, or IPTV Smarters.
2. Load [guide.xml](guide.xml) in any player that supports external EPG sources.
3. Open [index.html](index.html) in a browser for a simple preview page.
